PRACTICA A ENTREGAR:  war con codigo fuente y el archivo de base de datos.

Proyecto para portal de anuncios, donde cualquier usuario pueda registrar un 
                   producto de segunda mano a vender.

- Los anuncios tienen que tener 5 campos, a parte del ID, email y telefono
- Los productos registrados deben aparecer del último al primero.
- Separar las sql para que esten en una clase de constantes
- Buscador en el listado del cliente y zona admin
- Zona de admministración protegida por un filtro, donde el administrador 
    pueda borrar y editar anuncios.
- Categorias en zona usuario y zona admin
- Select, checkBox, radioButton
- Subida de imagen y muestreo en el listado de usuario y admin
- Paginación en zona usuario y admin
- Validaciones: 
	- Con campos vacios da mensaje de error, pero mantiene los campos correctos.
	- Validaciones los integers y doubles
	- Permitir comas en precio
	- Regex email y teléfono
